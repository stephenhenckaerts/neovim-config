# neovim-config by stephenhenckaerts

This repository contains my personal NeoVim configuration, streamlined for a modern development experience. It offers enhanced visuals, advanced navigation, comprehensive code manipulation capabilities, utilities, LSP & autocompletion support, and efficient version control.

## 📦 Plugins Used

### 1. **Core**
- [packer.nvim](https://github.com/wbthomason/packer.nvim)

### 2. **UI**
- [rose-pine](https://github.com/rose-pine/neovim) - A soothing color theme.

### 3. **File & Code Navigation**
- [telescope.nvim](https://github.com/nvim-telescope/telescope.nvim) - Advanced file and code searching.
- Dependencies: [plenary.nvim](https://github.com/nvim-lua/plenary.nvim)

### 4. **Code Parsing & Manipulation**
- [nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter) - Parser generator tool and incremental parsing for NeoVim.
- [nvim-treesitter/playground](https://github.com/nvim-treesitter/playground) - Playground for treesitter.

### 5. **Utilities**
- [plenary.nvim](https://github.com/nvim-lua/plenary.nvim) - Utility functionalities.
- [harpoon](https://github.com/ThePrimeagen/harpoon) - Bookmark and navigate to your most used files in vim.
- [undotree](https://github.com/mbbill/undotree) - Undo tree visualizer.
- [vim-fugitive](https://github.com/tpope/vim-fugitive) - Git wrapper.
- [lualine.nvim](https://github.com/nvim-lualine/lualine.nvim) - Statusline plugin.
- Dependencies: [nvim-web-devicons](https://github.com/nvim-tree/nvim-web-devicons)

### 6. **LSP & Autocompletion**
- [lsp-zero.nvim](https://github.com/VonHeikemen/lsp-zero.nvim) - Language Server Protocol tools.
- Dependencies:
  - [nvim-lspconfig](https://github.com/neovim/nvim-lspconfig) - Common configurations for built-in LSP client.
  - [mason.nvim](https://github.com/williamboman/mason.nvim) - Mason LSP.
  - [mason-lspconfig.nvim](https://github.com/williamboman/mason-lspconfig.nvim) - Mason LSP Configuration.
  - [nvim-cmp](https://github.com/hrsh7th/nvim-cmp) - Completion engine.
  - [cmp-nvim-lsp](https://github.com/hrsh7th/cmp-nvim-lsp) - LSP source for nvim-cmp.
  - [LuaSnip](https://github.com/L3MON4D3/LuaSnip) - Snippets solution.

### 7. **Git & Version Control**
- [copilot.vim](https://github.com/github/copilot.vim) - GitHub Copilot extension for Vim.
