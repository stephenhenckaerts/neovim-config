# neovim-config by stephenhenckaerts

This repository contains my personal NeoVim configuration, streamlined for a modern development experience. It offers enhanced visuals, advanced navigation, comprehensive code manipulation capabilities, utilities, LSP & autocompletion support, and efficient version control.

## 📦 Plugins Used

### 1. **Core**
- [packer.nvim](https://github.com/wbthomason/packer.nvim) - Essential for managing your NeoVim plugins.

### 2. **UI & Appearance**
- [onedark.vim](https://github.com/joshdick/onedark.vim) - A vibrant and elegant color scheme for NeoVim.

### 3. **File & Code Navigation**
- [NERDTree](https://github.com/preservim/nerdtree) - Tree explorer tool for navigating the filesystem.
- [telescope.nvim](https://github.com/nvim-telescope/telescope.nvim) - Advanced file and code searching.
  - Dependencies: [plenary.nvim](https://github.com/nvim-lua/plenary.nvim)
- [leap.nvim](https://github.com/ggandor/leap.nvim) - Efficient motion in the vim spirit.
  - Dependencies: [vim-repeat](https://github.com/tpope/vim-repeat)
- [harpoon](https://github.com/ThePrimeagen/harpoon) - Bookmark and navigate to your most used files in vim.

### 4. **Code Parsing, Highlighting & Manipulation**
- [nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter) - Parser generator tool and incremental parsing for NeoVim.
- [nvim-treesitter/playground](https://github.com/nvim-treesitter/playground) - Interactive interface to experiment with treesitter.
- [auto-pairs](https://github.com/jiangmiao/auto-pairs) - Automatic closing of quotes, parenthesis, brackets, etc.

### 5. **Utilities**
- [undotree](https://github.com/mbbill/undotree) - Visualize your undo history.
- [lualine.nvim](https://github.com/nvim-lualine/lualine.nvim) - A blazing fast and extensible status line.
  - Dependencies: [nvim-web-devicons](https://github.com/nvim-tree/nvim-web-devicons)
- [copilot.vim](https://github.com/github/copilot.vim) - GitHub Copilot extension for Vim.

### 6. **LSP & Autocompletion**
- [coc.nvim](https://github.com/neoclide/coc.nvim) - Conquer of Completion for Neovim, an intellisense engine.

### 7. **Git & Version Control**
- [vim-fugitive](https://github.com/tpope/vim-fugitive) - Git wrapper and commands within vim.
